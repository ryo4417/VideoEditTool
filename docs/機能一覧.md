# VideoEditTool 機能一覧（検証マトリクス）

現時点の全機能を4グループに整理。各機能の「狙い（意図した挙動）」を記載。
手分けして「狙い通り使えるか」を検証し、外れているものはバグとして修正する。

---

## グループ1: 入力・取り込み・プロジェクト
- **お試し動画生成** (`/api/make_sample`): 無音ギャップ入りサンプルを生成しリスト追加
- **ファイル選択**（複数可 `<input multiple>`）/ **ドラッグ&ドロップ**（複数可, `/api/upload`）
- **パス入力**（前後の引用符・空白を自動除去 `_clean_path`）
- **YouTube等URL取り込み** (`/api/fetch_url`, yt-dlp): DLして動画として追加
- **複数動画プロジェクト**: リストで切替。動画ごとに解析結果・手編集を独立保持
- **追加時の非割り込み**: 編集中の動画があれば新規追加で表示を奪わない（未選択時のみ表示）

## グループ2: 解析・ルール・文字起こし
- **検出ルール**（configでON/OFF）:
  - `silence`（無音を丸ごと）/ `tempo`（長い間の超過分のみ・自然な間は残す）
  - `filler`（フィラー語・複数語句可）/ `duplicate`（連続反復）/ `restate`（言い直し=類似句）
- **文字起こし**（ローカルWhisper, faster-whisper）: 内容ルール選択 or「文字起こし表示」で実行
  - GUIで **言語**（自動/日/英）・**精度**（tiny/base/small/medium/large-v3）選択, VADで幻聴抑制, モデルキャッシュ
- **案件プロファイル**（youtube/interview, 画面に説明表示）, deep-merge, スキーマ検証（型/未知キー/負値/不正format）
- **0件時の案内**（内容ルールが0件→理由＋文字起こし＋精度/言語ヒント）
- **背景解析**: 1本ずつ。表示中の動画のみオーバーレイ、他動画は編集継続可。完了で非割り込み

## グループ3: 編集UI（タイムライン・手編集・可視化）
- **タイムライン**: 緑=残す/赤=カット, 再生ヘッド, 波形背景
- **波形ズーム**（＋/−, 横スクロール, 同一時間軸で整合）
- **単語トラック**: 波形下に文字起こしを時間軸配置, カット語は取り消し線, クリックで頭出し
- **手編集**: 赤帯ドラッグ移動 / 端で伸縮 / ダブルクリック削除 / 空きドラッグで新規 / 「＋カット追加」
  / 一覧で秒数微調整・ON/OFF・✕削除。各カットに対応する文字起こし表示
- **Undo（↩元に戻す）** / **自動カットに戻す** / **自動検出（元）帯**（二段表示）
  / 自動検出帯を**ダブルクリックで個別復元**
- **編集後プレビュー**（有効カットを飛ばして再生）
- **解析中オーバーレイ**（経過秒・確実に消える）, 上書き防止の確認ダイアログ

## グループ4: 書き出し・CLI・その他
- **書き出し形式**: json / edl（映像V+音声AA・フレーム整合） / html（確認用） / fcpxml（有理数時刻） / 実カット動画（クリック音フェード）
- **解析1回→複数形式**（GUI/CLIとも）, **出力先フォルダ指定**, **Googleドライブ**（デスクトップ版・出力先制限）
- **CLI**: 複数入力（ワイルドカード, モデル使い回し）/ `--format a,b,c` / `--profile` / `--report` / `--render` / `--transcript` / `--output-dir` / UTF-8出力
- **AI品質採点**（ローカルLLM/Ollama, 任意・オフライン・接続失敗でも継続）
- **堅牢性**: no-cache配信, 解析タイムアウト, favicon 204, グレースフルなエラー（存在しない/非動画/未解析export/検出0）, Hostヘッダ検査
