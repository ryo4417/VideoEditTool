# VideoEditTool お試し・動作確認 手順書（チェックリスト）

上から順に試すと、ひととおりの機能を確認できます。
各項目の □ にチェックしながら進めてください。うまくいかない所があれば、その画面/エラーを控えて共有してください。

- 場所: `デスクトップ > VideoEditTool-試作機`
- 記号: 【操作】=あなたがやること / 【期待】=そうなればOK

---

## A. 起動（まず1回）

- [ ] **A-1 起動**
  - 【操作】フォルダ内の `start.bat` を**ダブルクリック**（または PowerShell で `powershell -ExecutionPolicy Bypass -File .\start.ps1`）
  - 【期待】黒い窓が出て準備が進み、ブラウザに編集画面（VideoEditTool — 編集確認）が開く
- [ ] **A-2 画面表示**
  - 【期待】上部に「動画ファイル」入力欄・点線ワク・「ファイルを選択」「解析」ボタン、検出チェック（無音/テンポ/フィラー/重複/言い直し）と言語が見える

---

## B. 動画を入れる（3通り）

- [ ] **B-1 お試し動画**
  - 【操作】「**お試し動画を作る**」ボタンを押す
  - 【期待】パス欄に `...\uploads\sample.mp4` が入り「解析を押してください」と表示
- [ ] **B-2 ドラッグ&ドロップ**
  - 【操作】手元の動画を点線ワクに**ドロップ**
  - 【期待】読み込み後、パス欄にパスが入る
- [ ] **B-3 パス貼り付け（クォート対策）**
  - 【操作】エクスプローラで動画を右クリック→「パスのコピー」→パス欄に貼る（前後に `"` が付く）
  - 【期待】解析時に前後の `"` が自動で外れ、正しく読み込める

---

## C. 自動カット（基本）

- [ ] **C-1 無音カット**
  - 【操作】「無音」だけチェック →「解析」
  - 【期待】動画の下に **緑（残す）/赤（カット）** の帯と波形が出る。無音区間が赤くなる
- [ ] **C-2 テンポ（間の短縮）**
  - 【操作】「テンポ」もチェック →「解析」
  - 【期待】長い「間」の超過分が赤くなる（間は少し残る）
- [ ] **C-3 削除率などの表示**
  - 【期待】サマリーに 総尺・残時間・削除率・カット数・残区間 が出る

---

## D. 文字起こし系ルール（発話のある動画で）

> 台本読みでない**雑談・vlog・実況**など、口ぐせのある動画が分かりやすいです。

- [ ] **D-1 言語選択**
  - 【操作】「言語」を動画に合わせて（日本語/英語/自動）選ぶ
- [ ] **D-2 フィラー除去**
  - 【操作】「フィラー」チェック →「解析」（初回は文字起こし準備で数分・経過秒が出る）
  - 【期待】「えー/あの/um」などが赤いカット候補になる。理由に「フィラー『…』」と出る
- [ ] **D-3 重複・言い直し**
  - 【操作】「重複」「言い直し」もチェック →「解析」
  - 【期待】繰り返し・言い直しが候補になる
- [ ] **D-4 0件のときの表示（空振り防止）**
  - 【操作】口ぐせの無い動画でフィラーを掛ける
  - 【期待】「該当箇所は見つかりませんでした」＋**文字起こし本文**が表示され、黙って空にならない
- [ ] **D-5 進捗表示**
  - 【期待】解析中はスピナーと「（N秒）」が回り、フリーズに見えない

---

## E. 手編集（自動カットの後、自分で微調整）★重要

- [ ] **E-1 移動**：赤帯を**ドラッグ**→位置が動き、削除率が変わる
- [ ] **E-2 伸縮**：赤帯の**左右の端をつまんでドラッグ**→長さが変わる
- [ ] **E-3 削除**：赤帯を**ダブルクリック**（または一覧の ✕）→消える
- [ ] **E-4 新規追加(ドラッグ)**：タイムラインの**何もない所を横にドラッグ**→新しい赤帯ができる
- [ ] **E-5 新規追加(ボタン)**：「**＋カット追加**」→再生位置に約1秒のカットが追加
- [ ] **E-6 数値微調整**：一覧で開始/終了の**秒数を直接入力**→帯が更新
- [ ] **E-7 個別ON/OFF**：一覧のチェックを外す→そのカットだけ無効化

---

## F. 確認と書き出し

- [ ] **F-1 頭出し**：候補の ▶ やタイムラインのクリックで再生位置が動く
- [ ] **F-2 編集後プレビュー**：「編集後プレビュー」ON で再生→**カットを飛ばして再生**され、仕上がりを耳で確認できる
- [ ] **F-3 書き出し（各形式）**：形式で json / edl / html / fcpxml を選んで「書き出し」→ `output/` にファイルができる
  - html を開くと、確認用の見やすいレポートが表示される
- [ ] **F-4 実カット動画**：「実カット動画も出力」ON で書き出し→編集済み mp4 ができ、尺が短くなっている
- [ ] **F-5 出力先/ Googleドライブ**：出力先フォルダを指定（Drive導入時は「Googleドライブに保存」ボタン）→そこに保存される

---

## G. プロファイル・エラー時の挙動

- [ ] **G-1 プロファイル**：「youtube」を選んで解析→無音の判定が強め（削除率が上がる）
- [ ] **G-2 存在しないパス**：でたらめなパスで解析→「ファイルが見つかりません」と日本語で出る
- [ ] **G-3 動画でないファイル**：テキスト等を指定→「動画として読めません（対応形式…）」と出る
- [ ] **G-4 検出0で解析**：全部チェックを外して解析→「検出項目が選ばれていません…」の案内が出る

---

## H. コマンド操作（任意・慣れた人向け）

PowerShell で（フォルダ内・`$env:PYTHONUTF8=1` を付けると日本語が綺麗に出ます）:

- [ ] **H-1 基本**：`python -m ui.cli sample.mp4`
- [ ] **H-2 複数形式まとめ（解析1回）**：`python -m ui.cli sample.mp4 --format fcpxml,edl,json --report`
- [ ] **H-3 実カット**：`python -m ui.cli sample.mp4 --render`
- [ ] **H-4 バッチ**：`python -m ui.cli *.mp4 --format edl --output-dir out`
- [ ] **H-5 文字起こし確認**：`python -m ui.cli sample.mp4 --transcript`
- [ ] **H-6 プロファイル/出力先**：`python -m ui.cli sample.mp4 --profile interview --output-dir "G:\マイドライブ\編集済み"`

## K. 追加機能（新）

- [ ] **K-1 複数動画**：動画を複数まとめてドロップ→リストに並ぶ→クリックで切替。各動画の解析・編集が**別々に保持**される
- [ ] **K-2 解析中の明示**：解析中は結果が薄くなりオーバーレイ＋経過秒。操作はロックされる
- [ ] **K-3 上書き防止**：編集した動画で再度「解析」→「編集を破棄して再解析しますか？」の確認が出る
- [ ] **K-4 元に戻す(Undo)**：カットを削除/移動した後「↩ 元に戻す」で戻る
- [ ] **K-5 自動カットに戻す**：手編集後「自動カットに戻す」で自動検出状態へ復元。下段の「自動検出（元）」帯で元との対比が見える
- [ ] **K-6 YouTube取り込み**：URL欄に動画URL→「URLから取り込み」→リストに追加（yt-dlp必要）
- [ ] **K-7 文字起こし精度**：「精度」を small→medium 等に上げて解析→認識精度が上がる（大きいほど低速）
- [ ] **K-8 プロファイル説明**：プロファイルを選ぶと画面に説明文が出る
- [ ] **K-9 波形ズーム**：波形右上の ＋/− で拡大縮小→横スクロールで細部を確認・編集できる
- [ ] **K-10 単語トラック**：波形の下に文字起こしが時間軸で並ぶ。カットに掛かる語は**取り消し線**。語クリックで頭出し
- [ ] **K-11 カット↔文字起こし対応**：カット一覧の各行に、その区間の発話（対応する文字起こし）が「…」表示される
