# VideoEditTool お試し・動作確認 手順書（チェックリスト）

上から順に試すと、ひととおりの機能を確認できます。
各項目の □ にチェックしながら進めてください。うまくいかない所があれば、その画面/エラーを控えて共有してください。

- 場所: `デスクトップ > VideoEditTool-試作機`
- 記号: 【操作】=あなたがやること / 【期待】=そうなればOK

---

## まず10分の最短コース（ここだけでも体験できます）

細かい確認は後回しで、ひととおりの流れをまず1回通してみるコースです。

- [ ] **1. 起動**：`start.bat` をダブルクリック → ブラウザに編集画面が開く
- [ ] **2. お試し動画**：「**お試し動画を作る**」を押す → パス欄に `sample.mp4` が入る
- [ ] **3. 無音＋テンポで解析**：検出の「無音」「テンポ（間の短縮）」にチェック →「解析」
  - → 動画の下に **緑（残す）/赤（カット）** の帯と波形が出る
- [ ] **4. 波形ズームで確認**：波形右上の ＋ で拡大し、横スクロールで赤帯の切れ目を細かく見る
- [ ] **5. 赤帯を手編集**：赤帯を**ドラッグで移動**、端を**つまんで伸縮**、いらない帯は**ダブルクリックで削除**
- [ ] **6. 編集後プレビュー**：「編集後プレビュー」を ON にして再生 → **赤帯を飛ばして再生**され、仕上がりが耳で分かる
- [ ] **7. 実カット書き出し**：「実カット動画も出力」を ON にして「書き出し」→ `output/` に短くなった mp4 ができる

ここまで通れば基本はOKです。以降は機能ごとの詳細チェックです。

## A. 起動（まず1回）

- [ ] **A-1 起動**
  - 【操作】フォルダ内の `start.bat` を**ダブルクリック**（または PowerShell で `powershell -ExecutionPolicy Bypass -File .\start.ps1`）
  - 【期待】黒い窓が出て準備が進み、ブラウザに編集画面（VideoEditTool — 編集確認）が開く
- [ ] **A-2 画面表示**
  - 【期待】上部に「動画ファイル」入力欄・点線ワク・「ファイルを選択」「解析」ボタン、検出チェック（無音/テンポ/フィラー/重複/言い直し）、言語・精度・プロファイルの選択が見える

---

## B. 動画を入れる（いろいろな入れ方）

- [ ] **B-1 お試し動画**
  - 【操作】「**お試し動画を作る**」ボタンを押す
  - 【期待】パス欄に `...\uploads\sample.mp4` が入り「解析を押してください」と表示
- [ ] **B-2 ドラッグ&ドロップ**
  - 【操作】手元の動画を点線ワクに**ドロップ**
  - 【期待】読み込み後、パス欄にパスが入る
- [ ] **B-3 ファイルを選択**
  - 【操作】「ファイルを選択」ボタンから動画を選ぶ
  - 【期待】パス欄にパスが入る
- [ ] **B-4 パス貼り付け（クォート対策）**
  - 【操作】エクスプローラで動画を右クリック→「パスのコピー」→パス欄に貼る（前後に `"` が付く）
  - 【期待】解析時に前後の `"` が自動で外れ、正しく読み込める
- [ ] **B-5 複数動画をまとめて投入**
  - 【操作】動画を**複数まとめてドロップ**（または続けて追加）
  - 【期待】リストに並び、クリックで切替できる。各動画の解析結果・編集内容が**別々に保持**される
- [ ] **B-6 URLから取り込み（YouTube等）**
  - 【操作】URL欄に動画URLを入れて「**URLから取り込み**」
  - 【期待】ダウンロード後、リストに追加される（yt-dlp が必要。未導入なら日本語で案内が出る）

---

## C. 自動カット（検出項目・言語・精度）

> 台本読みでない**雑談・vlog・実況**など、口ぐせのある動画が分かりやすいです（フィラー/重複/言い直しの確認に向く）。

- [ ] **C-1 無音カット**
  - 【操作】「無音」だけチェック →「解析」
  - 【期待】動画の下に **緑（残す）/赤（カット）** の帯と波形が出る。無音区間が赤くなる
- [ ] **C-2 テンポ（間の短縮）**
  - 【操作】「テンポ」もチェック →「解析」
  - 【期待】長い「間」の超過分が赤くなる（自然な間は少し残る）
- [ ] **C-3 言語選択**
  - 【操作】「言語」を動画に合わせて（自動/日本語/英語）選ぶ
- [ ] **C-4 フィラー除去**
  - 【操作】「フィラー」チェック →「解析」（初回は文字起こし準備で数分・経過秒が出る）
  - 【期待】「えー/あの/um」などが赤いカット候補になる。理由に「フィラー『…』」と出る
- [ ] **C-5 重複・言い直し**
  - 【操作】「重複」「言い直し」もチェック →「解析」
  - 【期待】繰り返し・言い直しが候補になる
- [ ] **C-6 精度（tiny〜large-v3）**
  - 【操作】「精度」を small→medium 等に上げて解析
  - 【期待】文字起こしの認識精度が上がる（大きいほど低速・初回DLが大きい）
- [ ] **C-7 削除率などの表示**
  - 【期待】サマリーに 総尺・残時間・削除率・カット数・残区間 が出る

---

## D. 解析中の挙動・上書き防止

- [ ] **D-1 解析中の明示（オーバーレイ＋ロック）**
  - 【期待】解析中は結果が薄くなりオーバーレイ＋経過秒（N秒）が出る。操作はロックされフリーズに見えない
- [ ] **D-2 上書き防止の確認ダイアログ**
  - 【操作】手編集した動画で再度「解析」を押す
  - 【期待】「編集を破棄して再解析しますか？」の確認が出る（キャンセルすれば編集は保持される）
- [ ] **D-3 0件のときの表示（空振り防止）**
  - 【操作】口ぐせの無い動画でフィラーを掛ける
  - 【期待】「該当箇所は見つかりませんでした」＋**文字起こし本文**が表示され、黙って空にならない

---

## E. 波形ズーム・単語トラック・対応表示

- [ ] **E-1 波形ズーム**
  - 【操作】波形右上の ＋/− で拡大縮小
  - 【期待】横スクロールで細部を確認・編集できる
- [ ] **E-2 単語トラック**
  - 【期待】波形の下に文字起こしが時間軸で並ぶ。カットに掛かる語は**取り消し線**。語をクリックすると頭出し
- [ ] **E-3 カット↔文字起こし対応**
  - 【期待】カット一覧の各行に、その区間の発話（対応する文字起こし）が「…」表示される

---

## F. 手編集（自動カットの後、自分で微調整）★重要

- [ ] **F-1 移動**：赤帯を**ドラッグ**→位置が動き、削除率が変わる
- [ ] **F-2 伸縮**：赤帯の**左右の端をつまんでドラッグ**→長さが変わる
- [ ] **F-3 削除**：赤帯を**ダブルクリック**（または一覧の ✕）→消える
- [ ] **F-4 新規追加(ドラッグ)**：タイムラインの**何もない所を横にドラッグ**→新しい赤帯ができる
- [ ] **F-5 新規追加(ボタン)**：「**＋ カット追加**」→再生位置に約1秒のカットが追加
- [ ] **F-6 数値微調整**：一覧で開始/終了の**秒数を直接入力**→帯が更新
- [ ] **F-7 個別ON/OFF**：一覧のチェックを外す→そのカットだけ無効化
- [ ] **F-8 元に戻す(Undo)**：削除/移動した後「**↩ 元に戻す**」で直前の編集が戻る
- [ ] **F-9 自動カットに戻す**：手編集後「**自動カットに戻す**」で自動検出状態へ復元
- [ ] **F-10 二段表示（元との対比）**：下段の「**自動検出（元）**」帯で、手編集後と自動検出時の対比が見える

- [ ] **G-2 編集後プレビュー**：「**編集後プレビュー**」ON で再生→**カット区間を飛ばして再生**され、仕上がりを耳で確認できる

## H. 書き出し

- [ ] **H-1 書き出し（各形式）**：形式で json / edl / html（確認用）/ fcpxml を選んで「書き出し」→ `output/` にファイルができる
  - html を開くと、確認用の見やすいレポートが表示される
- [ ] **H-2 実カット動画**：「**実カット動画も出力**」ON で書き出し→編集済み mp4 ができ、尺が短くなっている
- [ ] **H-3 出力先フォルダ**：出力先フォルダを指定（空欄なら `output/`）→そこに保存される
- [ ] **H-4 Googleドライブ**：Drive導入時は「**Googleドライブに保存**」ボタン（または同期フォルダを出力先に指定）→Driveに保存される

## I. プロファイル・エラー時の挙動

- [ ] **I-1 プロファイル**：「youtube」を選んで解析→無音の判定が強め（削除率が上がる）

## J. コマンド操作（任意・慣れた人向け）

PowerShell で（フォルダ内・`$env:PYTHONUTF8=1` を付けると日本語が綺麗に出ます）:

- [ ] **J-1 基本**：`python -m ui.cli sample.mp4`
- [ ] **J-2 複数形式まとめ（解析1回）**：`python -m ui.cli sample.mp4 --format fcpxml,edl,json --report`
- [ ] **J-3 実カット**：`python -m ui.cli sample.mp4 --render`
- [ ] **J-4 複数入力（バッチ）**：`python -m ui.cli *.mp4 --format edl --output-dir out`
- [ ] **J-5 文字起こし確認**：`python -m ui.cli sample.mp4 --transcript`
- [ ] **J-6 プロファイル/出力先**：`python -m ui.cli sample.mp4 --profile interview --output-dir "G:\マイドライブ\編集済み"`
