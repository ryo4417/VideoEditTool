# VideoEditTool

動画編集ツール（技術スタック未定）。

## 概要
- 目的: 動画の編集・変換・処理を行うツール。
- ステータス: 初期セットアップ段階。言語・フレームワークは未確定。
- 想定機能（候補）: トリミング / 結合 / フォーマット変換 / バッチ処理（ffmpeg 連携を想定）。

## ディレクトリ構成
```
VideoEditTool/
├── src/      # ソースコード
├── tests/    # テスト
├── docs/     # ドキュメント
├── assets/   # サンプル・アイコン等の静的ファイル
├── CLAUDE.md
└── .gitignore
```

## 開発方針
- 技術スタックを決定したら、このファイルにビルド/実行/テストのコマンドを追記する。
- 大きな動画ファイル・出力物はコミットしない（`.gitignore` 参照）。

## Git
- ブランチ: `main`
- コミットメッセージは変更内容が分かる簡潔な日本語または英語で記述する。
